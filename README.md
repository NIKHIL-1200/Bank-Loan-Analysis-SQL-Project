# Bank Loan Analysis Using SQL

## Project Overview

This project analyzes a bank loan dataset using SQL. The goal is to find useful business insights related to loan applications, loan amount, loan status, customer income, credit score, interest rate, home ownership, loan purpose, and high-risk applicants.

## Dataset Columns Used

* person_age
* person_gender
* person_education
* person_income
* person_emp_exp
* person_home_ownership
* loan_amnt
* loan_intent
* loan_int_rate
* loan_percent_income
* cb_person_cred_hist_length
* credit_score
* previous_loan_defaults_on_file
* loan_status

## Tools Used

* MySQL
* MySQL Workbench
* GitHub

## Key Analysis Questions

1. What is the total number of loan applications?
2. What is the loan status breakdown?
3. What is the overall loan approval rate?
4. What is the total loan amount requested?
5. What is the average loan amount requested?
6. What is the average interest rate?
7. What is the average credit score of applicants?
8. Which loan purposes have the highest number of applications?
9. Which loan purposes have the highest approval rate?
10. How does home ownership affect loan applications?
11. How does home ownership affect loan approval?
12. How does education level affect loan approval?
13. How does gender affect loan approval?
14. How do previous loan defaults affect loan approval?
15. Which age groups have higher loan approval rates?
16. How does income group affect loan approval?
17. How does credit score group affect loan approval?
18. Which applicants can be considered high-risk loan applicants?

## Project Files

* `Insights.md` - Contains business questions, results, and insights.
* `bank_loan_analysis.sql` - Contains all SQL queries used for analysis.
* `Screenshots` - Contains screenshots of SQL query outputs.


## Business Insights

The analysis helps understand bank loan application patterns, customer financial behavior, loan approval trends, and risk factors.

The project provides insights into how income, credit score, education, home ownership, loan purpose, previous defaults, and loan-to-income ratio can affect loan approval and applicant risk.

## Conclusion

This project helped me practice SQL concepts such as aggregation, grouping, filtering, ordering, conditional logic, and business analysis using real-world banking data.

The project also helped me understand how SQL can be used in the banking domain to analyze loan applications, approval trends, customer profiles, and high-risk applicants.
